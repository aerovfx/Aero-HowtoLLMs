# Kiểm Tra Phiên Bản

## Các Phiên Bản Được Sử Dụng Trong Khóa Học

Khóa học này sử dụng các phiên bản sau:

| Thư viện | Phiên bản |
|----------|-----------|
| Python | 3.10.12 |
| Pandas | 1.5.3 |
| Scikit-learn | 1.2.2 |
| NumPy | 1.23.5 |
| TensorFlow | 2.15 |
| Keras | 2.15 |
| Transformers | 4.35.2 |

**Lưu ý quan trọng:** Transformers 4.35.2 là phiên bản rất quan trọng.

Các phiên bản này được cài đặt sẵn trên Google Colab, nền tảng chúng ta sẽ sử dụng cho các demo.

## Yêu Cầu Tối Thiểu

Tuy nhiên, khóa học này có thể áp dụng cho bất kỳ các phiên bản sau:

| Thư viện | Phiên bản tối thiểu |
|----------|---------------------|
| Python | > 3.6 |
| Pandas | > 1.0 |
| Scikit-learn | > 1.0 |
| NumPy | > 1.0 |
| TensorFlow | > 2.0 |
| Keras | > 2.0 |
| Transformers | > 4.0 |
